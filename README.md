# Example project for contract.dev

Simple Solidity project for demonstrating contract.dev's capabilities.
You can list NFTs for sale on marketplace for an amount of ERC20.
Truly amazing.

## Contracts

This project includes three simple contracts:

- **ContractDevToken** - An ERC20 token used for marketplace transactions
- **ContractDevNFT** - An ERC721 NFT contract for minting NFTs
- **ContractDevMarketplace** - A marketplace where users can list and purchase NFTs using the token
